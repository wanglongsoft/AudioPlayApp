# AudioPlayApp
基于Android平台音频采集的简单教程，包含音频采集，音频播放，音频文件(PCM)转WAV文件
#### 编译环境
AndroidStudio:3.6.3
#### 功能点
* AudioRecord 采集音频数据
* AudioTrack 播放音频文件
